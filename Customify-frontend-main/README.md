Customify Frontend 🛍️
A modern, high-performance e-commerce interface built with Vue.js. This application provides a fluid, reactive user experience for browsing products, managing a shopping cart, and secure checkout.

🚀 Tech Stack
Framework: Vue.js 3 (Composition API)

Styling: Tailwind CSS (Utility-first CSS)

Routing: Vue Router (Single Page Application navigation)

State Management: Reactive ref/reactive (or Pinia/Vuex if applicable)

HTTP Client: Fetch API / Axios

✨ Key Features
Responsive Storefront: Mobile-first design that looks great on any device.

Dynamic Product Feed: Real-time data fetching from the backend /api/products endpoints.

Category Filtering: Instant view updates using Vue Router params for different product categories.

Secure Auth Integration: Dedicated views for Login and Registration that store and send JWT tokens for protected actions.

Interactive Cart: Add, remove, and update quantities with real-time total price calculation.
